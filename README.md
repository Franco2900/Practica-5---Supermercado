# Practica-5---Supermercado
🛒 Supermercado Online

Proyecto completo de un sistema de supermercado online con frontend en React y backend en Node.js + MySQL.
Permite a los usuarios registrarse, iniciar sesión, agregar productos y ofertas al carrito, modificar cantidades, realizar compras y vaciar el carrito. Los usuarios con el rol de admin tienen más funciones.

✨ Características principales para usuarios

Carrito dinámico con soporte para productos individuales y ofertas.
Gestión de cantidades: actualización en tiempo real de productos y ofertas.
Favoritos: posibilidad de marcar productos como favoritos para acceder más rápido.
Historial de compras: listado de todas las compras realizadas.
Filtros por fecha: búsqueda de compras pasadas según rango temporal.
Descarga de comprobantes: generación de archivo PDF para cada compra realizada.
Notificaciones (toasts) para confirmar acciones del usuario.
Context API para manejar estado global de usuario y carrito.

🔧 Funcionalidades de administrador

Gestión de productos y ofertas: crear, modificar y eliminar.
Gestión de roles: cambiar el rol de los usuarios (de cliente a admin o viceversa).
Reportes: visualización de estadísticas y reportes sobre productos y ofertas.

🛠️ Tecnologías utilizadas

Frontend: React, Context API, Bootstrap, FontAwesome.
Backend: Node.js, Express, MySQL.
Base de datos: MySQL con tablas para usuarios, productos, ofertas, carritos, favoritos e historial de compras.

🚀 Objetivo

Este proyecto busca simular un sistema de compras en línea para un supermercado, mostrando cómo integrar frontend y backend, con un flujo completo desde la selección de productos hasta la compra final, incluyendo funcionalidades de gestión y administración.
